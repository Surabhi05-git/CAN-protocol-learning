# CAN Message and Signal Extraction Using DBC

This repository contains presentation slides prepared to understand
CAN message structure and signal extraction using DBC files.

## Topics Covered
- CAN protocol basics
- Standard CAN frame format
- Role of DLC and padding
- Signals vs CAN messages
- DBC file structure
- Bit extraction using start bit and length
- Scaling, offset, and physical value calculation
- Example CAN signal decoding from DBC

## Tools Referenced
- Vector CANdb++
- DBC files
- Classical CAN

## Purpose
This PPT was created for academic explanation and self-learning
in automotive communication systems.
